# Nuxt Taurify

## Prerequisites
 - Follow the link https://tauri.app/v1/guides/getting-started/prerequisites/

## Installation

```bash
# Using NPM
npm install

# Using Yarn ** Hihgly Recommended **
yarn install # or just 'yarn'
```

## Build

